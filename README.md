# Tailwind CSS — 4-Day Course Syllabus

A complete four-day syllabus designed to teach Tailwind CSS from fundamentals to advanced component building.

---

## DAY 1 — FUNDAMENTALS & LAYOUT
### Topics Covered
- **Introduction to Tailwind CSS**
- **Utility-First Concept**
- **Setup and Configuration**
- **Container and Spacing**
  - Margin & Padding utilities
  - Width & Height utilities
- **Flexbox Layout**
  - Flex direction, wrap, order
  - Justify-content & align-items
- **Grid System**
  - Grid templates & columns
  - Gap & placement
- **Box Sizing & Display**
  - block, inline-block, inline, hidden
- **Positioning**
  - relative, absolute, fixed, sticky
- **Z-index**

---

## DAY 2 — TYPOGRAPHY & BACKGROUNDS
### Topics Covered
- **Text Properties**
  - Font size, weight, family
  - Text alignment & transformation
  - Letter spacing & line height
- **Colors & Opacity**
  - Text colors
  - Background colors
  - Border colors
  - Opacity utilities
- **Background Properties**
  - Background size, position, repeat
  - Gradient backgrounds
- **Sizing**
  - Min/Max width & height
  - Object fit & position
- **Visibility**
  - visible, invisible
  - screen-reader-only utilities

---

## DAY 3 — BORDERS & EFFECTS
### Topics Covered
- **Borders**
  - Border width, style, color
  - Border radius
- **Shadows**
  - Box shadows (all levels)
  - Inner shadows
- **Effects & Filters**
  - Opacity & blend modes
  - Blur, brightness, contrast
  - Grayscale, hue-rotate, invert
  - Saturate, sepia
- **Transitions & Animations**
  - Transition property, duration, timing
  - hover, focus, active states
  - Transform utilities (scale, rotate, translate)
  - Animation utilities

---

## DAY 4 — RESPONSIVE DESIGN & COMPONENTS
### Topics Covered
- **Responsive Design**
  - Breakpoints (sm, md, lg, xl, 2xl)
  - Mobile-first workflow
  - Responsive utilities
- **Interactive Components**
  - Buttons (all states)
  - Cards & containers
  - Navigation bars & menus
- **Forms**
  - Input fields & styling
  - Checkboxes & radio buttons
  - Form layout & validation states
- **Component Examples**
  - Common UI patterns
  - Best practices for component structure
- **Project Structure & Organization**
  - Component extraction
  - Using `@apply` directive
  - Customizing Tailwind configuration

---

## OPTIONAL / ADVANCED TOPICS
- Dark mode implementation
- Customizing the default theme
- Creating custom utilities
- Performance optimization
- Framework integration (React, Vue, etc.)
- Deployment considerations

---

## PRACTICAL EXERCISES
- **Day 1:** Build a responsive layout (header, sidebar, main content)
- **Day 2:** Style a blog post using typography & backgrounds
- **Day 3:** Build interactive cards with hover effects & animations
- **Day 4:** Create a complete responsive multi-component website
