Introduction to Tailwind CSS

Tailwind CSS is a utility-first CSS framework that allows you to build custom designs without ever leaving your HTML.

Key Benefits:

  • No more custom CSS to maintain
  • Responsive design made easy
  • Highly customizable
  • Small bundle size with PurgeCSS
  • Consistent design system

Setup & Installation

CDN Method (Quick Start)

<script src="https://cdn.tailwindcss.com"></script>

npm Installation (Recommended)

npm install -D tailwindcss
npx tailwindcss init

Then add Tailwind directives to your CSS:

@tailwind base;
@tailwind components;
@tailwind utilities;
